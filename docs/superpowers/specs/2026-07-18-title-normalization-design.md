# 一级标题自动补全设计

## 目标

消除模型偶发遗漏 Markdown 一级标题导致的格式性失败，同时保持所有证据、来源链接和高风险免责声明的质量门槛。

## 行为

在调用 `validate_generated_content` 前执行标题规范化：

- 输入已有 `# ` 一级标题时，内容不变。
- 输入缺少一级标题时，取第一个非空行作为标题；去除常见 Markdown 强调符号，并在正文前插入 `# <标题>`。
- 仅当找不到任何非空文本时保持原样，由现有质量校验报错。
- 规范化后的文章继续经过现有引用、参考来源、链接、禁用表达和分类免责声明检查；此改动不允许绕过任何这些检查。

## 可观测性与验证

- 标题被补全时记录一个不包含正文的诊断事件。
- 单元测试覆盖：保留已有标题、从首行补标题、空内容仍被拒绝、补题后其它质量问题仍会失败。
